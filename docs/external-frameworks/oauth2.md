# OAuth 2.0

## Evidence posture
`sourced_intake` — RFC 6749 captured; supplied enterprise-governance article retained as secondary context only.

## Published scope
OAuth 2.0 is an authorization framework enabling a client to obtain limited access to protected resources on behalf of a resource owner or itself.

Canonical source: https://www.rfc-editor.org/rfc/rfc6749

Secondary context supplied: https://nhimg.org/faq/how-should-security-teams-govern-oauth-20-in-enterprise-environments/
