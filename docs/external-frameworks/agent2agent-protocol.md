# Agent2Agent Protocol

## Evidence posture
`sourced_intake` — canonical protocol specification captured; no agent-runtime observation claimed.

## Published scope
A2A defines agent interoperability concepts for discovery through Agent Cards, messages, tasks, artifacts, completion, failure, and authenticated interactions.

Canonical source: https://a2a-protocol.org/latest/specification/

## StegVerse relationship
`adjacent`

Agent discovery and task exchange can enter the StegVerse path as proposed interaction evidence. Delegation, current standing, and consequence authority remain separately evaluated.

```text
agent discovered != trusted for every action
task accepted != consequence authorized
task completed != transition admissible
```
