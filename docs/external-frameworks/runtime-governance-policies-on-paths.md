## Definition

Runtime Governance for AI Agents: Policies on Paths presents runtime evaluation as a governance approach for AI-agent execution paths.

## Relationship To Admissibility

Runtime path governance is useful as a comparison target for runtime checks, partial execution paths, proposed next actions, policy state, and governance at execution time.

It does not replace commit-time admissibility review, execution authority checks, receipt-bound execution, or Admissible-Existence formalism sources.
