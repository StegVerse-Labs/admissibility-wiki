# W3C PROV

## Evidence posture
`sourced_intake` — official W3C PROV recommendation source captured. The supplied Wikipedia page is retained only as a discovery reference, not as canonical evidence.

## Published scope
W3C PROV defines a provenance model and related specifications for representing entities, activities, agents, generation, usage, derivation, and attribution.

Canonical source: https://www.w3.org/TR/prov-overview/

Discovery reference supplied: https://en.wikipedia.org/wiki/W3C_Prov

## StegVerse relationship
`adjacent`

PROV records can support reconstruction of who or what acted, which activities occurred, and how artifacts were derived. Provenance representation does not prove legitimacy, current authority, or admissibility.

## Benchmark relevance
`reconstruction_boundary`, `evidence_freshness_boundary`, `authority_boundary`, `interoperability_path`

## Non-claims
Provenance is not permission. Attribution is not current delegation. A reconstructable chain is not automatically an admissible chain.
