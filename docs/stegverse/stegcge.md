# StegCGE

StegCGE is the StegVerse governance engine concept for managing decisions and enforcement around declared actions.

## Working Boundary

A useful separation is:

- TV/TVC provides authority and evidence inputs;
- StegCGE manages decisions and enforcement;
- repositories act by declaring actions.

## Core Question

StegCGE should answer:

> Is this declared action admissible under the current authority, policy, evidence, and context state?
