# Transition Table

The Transition Table is a StegVerse formalism surface for classifying proposed state changes and their governance posture.

## Purpose

The table should help make transition review explicit.

It can represent:

- transition type;
- authority class;
- policy reference;
- evidence posture;
- review posture;
- admissibility result;
- commit-time validity;
- receipt status.

## Why It Matters

The table makes it harder to confuse animation, visibility, or execution with governance.

A visible transition is not automatically an admissible transition.
