# Governance Boundary

A governance boundary is the declared limit of what a system claims to govern, observe, authorize, enforce, or prove.

Clear boundaries matter because systems often appear to govern more than they actually govern.

## Boundary Questions

A governance boundary should answer:

- what the system claims;
- what the system does not claim;
- which authority classes it recognizes;
- which evidence it accepts;
- where decision responsibility begins;
- where decision responsibility ends;
- which external systems it depends on.
