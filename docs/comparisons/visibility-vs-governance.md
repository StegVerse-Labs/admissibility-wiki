# Visibility vs Governance

Visibility means a transition or process can be seen.

Governance means the transition or process is constrained by declared authority, policy, evidence, review, and consequence rules.

## Core Distinction

Visibility is not governance.

A dashboard can show activity without controlling whether that activity had standing.

An animation can show state movement without proving that the movement was admissible.

## Practical Rule

A governed system should make the binding decision, authority basis, and receipt posture visible — not merely the motion of the system.
