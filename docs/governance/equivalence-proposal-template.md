# Equivalence Proposal Template

## Purpose

This template is used when proposing that an external term is equivalent, overlapping, adjacent, broader, narrower, contradictory, or unresolved relative to an Admissibility Wiki glossary term.

It prevents unsupported equivalence claims by requiring explicit evidence and a review posture before any glossary page is changed.

## Non-claims

A completed proposal does not by itself:

- accept the equivalence;
- update the glossary;
- prove public adoption;
- create admissibility standing;
- replace a decision record;
- replace executable formalism tests where proof is required.

## Evidence Requirements

A mature equivalence proposal must include:

1. At least one source for the external term.
2. A statement of whether the source is primary, secondary, standards-based, academic, vendor-authored, or community-authored.
3. A claim-by-claim comparison.
4. A mismatch section.
5. A risk-of-overclaiming section.
6. A proposed glossary placement.
7. A decision record link when reviewed.
